# Icons

## Rules

- Use a single icon system per project
- Icons must be consistent in size and style
- Icons should support text, not replace it

## Allowed Sources

- Bootstrap → Font Awesome https://fontawesome.com/
- Tabler → Tabler Icons https://tabler.io/icons
- Tailwind → Tailwind UI Assets https://tailwindcss.com/plus/ui-blocks/documentation/assets
