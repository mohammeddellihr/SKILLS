# Page Header

The top section of a content area that introduces the context of the page.

## Structure
- Title
- Subtitle (Optional)
- Actions (Optional)
- Breadcrumbs (Optional)

## Guidelines
- **Title**: The main heading (`h1` or primary title styling) of the current page. Keep it concise.
- **Subtitle**: Supporting text below the title. Example: "User #218" or descriptive text about the page's purpose.
- **Actions**: Primary and secondary buttons related to the page context.
  - *Create* (Primary action on list pages)
  - *Update / Edit* (On view/detail pages)
  - *Delete* (Destructive action, usually placed separately or styled dangerously)

## Examples
### List View
- Title: "Users"
- Actions: [Create New User] (Primary)

### Detail View
- Title: "John Doe"
- Subtitle: "Admin Account"
- Actions: [Edit Profile] (Secondary), [Suspend User] (Danger)
